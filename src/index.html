◎ Сложность

Меняется только из главного меню. Текущий прогресс кампании не сбрасывается: новые параметры применяются к дальнейшим секторам и дропу.

Нормальная

Режим «одна жизнь»: любая гибель модуля стирает прогресс. Сохранения только после 2-го и 4-го боссов (волны 8 и 16), чтобы можно было продолжить после перерыва; при смерти сейв обнуляется.

⌗ Схема

Клавиатура + мышь

Бой

  • WASD — движение
  • Мышь — прицел и стрельба (ЛКМ)
  • Shift — ускорение / варп (класс «Поток»)
  • Пробел — умение SPC
  • 1 / 2 / 3 — протоколы (ремонт, сброс КД, EMP)
Геймпад

Бой

  • LS — движение
  • RS — прицел (направление носа)
  • RT (R2) — стрельба
  • LB (L1) — ускорение / варп
  • RB (R1) — доп. атака
  • D-Pad ← / ↑ / → — протоколы 1 / 2 / 3
  • A (Cross) — подтвердить / выбрать
  • B (Circle) — назад / отмена
  • Start (Options) — пауза

Меню

  • LS / D-Pad — навигация
  • A — выбрать · B — назад
  • LB / RB — переключение вкладок

Подключите геймпад и нажмите любую кнопку для проверки.
