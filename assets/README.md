# assets/

Сюда кладём статические ресурсы игры.

## Иконки приложения (нужны для сборки)

Для electron-builder нужны иконки в трёх форматах:

| Файл | Платформа | Размер |
|---|---|---|
| icon.ico | Windows | минимум 256x256 (multi-size ICO) |
| icon.icns | macOS | минимум 512x512 |
| icon.png | Linux | минимум 512x512, формат PNG |

### Как создать иконки

1. Нарисуй изображение 1024×1024 пикселей (PNG)
2. Для Windows .ico — используй https://icoconvert.com
3. Для macOS .icns — используй `iconutil` на Mac или https://cloudconvert.com
4. Для Linux .png — просто сохрани 512×512 PNG

### Быстрый вариант без иконок

Если иконок нет, electron-builder использует стандартную иконку Electron.
Сборка пройдёт, но приложение будет выглядеть не брендированным.
