# notuse — не подключается рантаймом

Сюда перенесены файлы, на которые **нет ссылок** из `src/index.html`, `main.js`, `preload.js` и смежных скриптов (проверено grep).

| Что | Почему здесь |
|-----|----------------|
| `balance.json` | Баланс задаётся в `GAME_BALANCE_CONFIG` в `index.html`, JSON не загружается. |
| `assets-graf/*` | Дубликаты видео/PNG схем, неиспользуемые SVG/лого; splash использует только `Logostart.mp4` в `assets/graf/`; схемы управления — только `клавиатура разметка.svg` и `геймпад разметка.svg`. |

При необходимости вернуть файл в игру — скопировать обратно в `assets/` и добавить ссылку в код/HTML.
